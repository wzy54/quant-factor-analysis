# A股银行板块动量因子分析

## 项目简介

基于 Tushare Pro 数据，对 A 股银行板块进行动量因子有效性研究。
完整实现从数据获取、因子计算、IC 分析到分组回测的量化研究全流程。

## 核心结论

- 分析了 **27只**银行股，覆盖 **2019-2024** 年共 **1416** 个交易日
- 动量因子 IC 均值 = **-0.029**，ICIR = **-0.093**
- 银行板块存在显著**反转效应**：过去20日跌幅最大的股票，未来20日平均收益 **+0.478%**；涨幅最大的反而 **-0.105%**
- 分组收益单调递减，反转多空组合净值曲线整体向上

## 面试要点

**为什么动量因子在银行板块是反转效应？**

银行股属于典型价值股，股价受业绩和估值驱动，短期动量信号容易被均值回归抵消。
IC 负值说明因子方向与预期相反，这本身就是一个有价值的结论——
银行板块适合用**反转因子**而非动量因子构建 alpha。

**局限性与后续改进**

- 未扣除交易成本，实际收益会有所下降
- 仅用单一动量因子，后续可引入 PE、ROE 等基本面因子做多因子合成
- 可用 Barra 风险模型做行业中性化处理
